# 🐱 Escove o Gato

Um jogo simples, divertido e cheio de personalidade 💖

---

## 🎮 Sobre o jogo

**Escove o Gato** é um mini game onde você precisa escovar um gatinho…  
mas cuidado: **eles não gostam de ser incomodados 😾**

Seu objetivo é fazer o máximo de pontos possível escovando o gato sem ser pego!

---

## 💡 Como jogar

- 🖱️ **Mova o mouse** para posicionar a mão  
- ✊ **Segure o clique** para escovar o gato  
- ⭐ Cada movimento certo = pontos  
- 👀 Se o gato virar... **PARE IMEDIATAMENTE**  
- 😡 Se continuar escovando enquanto ele olha → **GAME OVER**
